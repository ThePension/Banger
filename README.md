# Banger

## Description

Fast notes taking language

### Features

- Titres de plusieurs niveaux
- Boucles
- Opérations (+, -, ...),
- Tests,
- Portée de variables,
- Couleur de la police
- Couleur de fond
- Bloc de code
- Extension VS Code pour preview
